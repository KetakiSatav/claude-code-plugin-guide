# Building a Claude Code Plugin — An Interactive Guide

A hands-on walkthrough for building your first Claude Code plugin — from empty folder to installed, working plugin with slash commands, hooks, and (optionally) agents, skills, or MCP servers.

## Read the guide

- **[Read online →](./claude-code-plugin-guide.md)** — renders in the browser with expandable code sections and clickable checkboxes.
- **[Download PDF →](./claude-code-plugin-guide.pdf)** — the same content, print-ready, all code sections expanded.

## What's inside

- What a Claude Code plugin is and what pieces it can contain
- Prerequisites and design questions to answer up front
- The four-file skeleton every plugin needs
- Adding slash commands, hooks, agents, skills, and MCP servers
- Registering a local marketplace for dev iteration (with the `/plugin marketplace add` gotcha most people hit)
- End-to-end verification and a debug playbook
- Publishing your plugin to others via git
- Reusable patterns and a v0.1.0 release checklist

## Who it's for

Developers who want to extend Claude Code with their own automations — greetings, shortcuts, hooks that integrate with their tools, or agents specialised to their codebase. No Claude Code plugin experience assumed; a working shell and basic Node knowledge are enough.
